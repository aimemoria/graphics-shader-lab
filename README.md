# Graphics and Shader Experiments

A collection of graphics and shader experiments covering OpenGL-style rendering exercises and Unity shader prototypes. Includes demo scenes, shader variants, and supporting assets.

## Contents
- Advanced shader prototypes and materials
- Demo scenes and rendering exercises
- Example source code for geometry and lighting

## Requirements
- C++ compiler (for native demos)
- Unity (for shader projects)

## Project structure
```

Putting It All Together/

YourSurroundWorld/
```
